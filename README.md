# PA5 实验报告：代码生成器

本仓库包含了 Cool 编译器 PA5 阶段（代码生成）的实现代码。该实验的主要任务是将 Cool 语言的抽象语法树 (AST) 转换为 MIPS 汇编代码。

## 文件说明

本项目主要修改了以下核心文件以实现功能：

*   **`cgen.cc`**
    *   核心实现文件。
    *   负责生成 MIPS 指令、管理符号表、处理继承关系以及方法的动态分发。
*   **`cgen.h`**
    *   头文件，定义了代码生成器所需的类结构。
*   **`cool-tree.h`**
    *   修改了 AST 节点的定义，添加了代码生成的接口。
*   **`cool-tree.handcode.h`**
    *   包含辅助 AST 节点进行代码生成的补充代码。

## 测试用例

*   **`stack.cl`**
    *   一个从PA3处获取的Cool语言测试程序，用于全面测试代码生成器的正确性，包括对象创建、方法调用、IO 输出等功能。
